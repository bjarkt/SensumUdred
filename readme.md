# SensumUdred
Semesterprojekt

## How to run
java and javac must be in path.
- windows:
    - double click `run.bat`
- linux/mac:
    - open terminal in project directory
    - run command: `./run.sh`
    
## External services
Using http server to simulate various external services, such as cpr register and eboks.
